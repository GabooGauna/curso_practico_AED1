## 📁 Práctica 3: Funciones Matemáticas y Composición

### 3.1. Programa de Funciones Básicas

Escribir un programa que contenga las siguientes funciones matemáticas:

- **a) mitadDe(n):** Devuelve la mitad de un número ($10 \rightarrow 5$).
- **b) dobleDe(n):** Devuelve el doble de un número ($5 \rightarrow 10$).
- **c) tripleDe(n):** Devuelve el triple de un número ($5 \rightarrow 15$).
- **d) anteriorDe(n):** Devuelve el número anterior al recibido ($10 \rightarrow 9$).
- **e) posteriorDe(n):** Devuelve el número posterior al recibido ($10 \rightarrow 11$).

> **f) Función de prueba:** El programa debe solicitar al usuario un número entero y mostrar los resultados de aplicar todas las funciones anteriores sobre ese valor.

---

### 3.3. Composición de Funciones

Agregar al programa anterior una función llamada `posteriorDelDobleDe()`.

**Consigna:**
La función debe devolver el posterior del doble del número recibido como parámetro.

- **Requisito:** Se deben invocar las funciones `dobleDe()` y `posteriorDe()` ya definidas anteriormente.
- **Ejemplo:** `posteriorDelDobleDe(7)` $\rightarrow$ $15$

> **Nota:** No olvides actualizar la función de prueba para incluir este nuevo resultado en la pantalla.
